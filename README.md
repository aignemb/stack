# Stack2

A TUI task manager built with Bubble Tea v2.

## Usage

### Navigation

- `j` - Move cursor down
- `k` - Move cursor up
- `J` - Swap current item with item below
- `K` - Swap current item with item above

### View Switching

- `s` - Switch to stack view
- `h` - Switch to heap view
- `a` - Switch to archive view

### Task Actions

- `n` - Create new task (stack/heap views only)
- `e` - Edit current task
- `d` - Delete current task (press twice to confirm)
- `p` - Pop/promote task (moves between stack and heap)
- `z` - Archive/unarchive (move to/from archive)

### Task Edit Mode

- `tab` - Switch between title and priority fields
- `enter` - Save and exit
- `esc` - Cancel and exit

### Global

- `q` - Quit (saves data)
- `ctrl+c` - Quit (saves data)

### Priority Levels

- `1` = High
- `2` = Medium (default)
- `3` = Low
